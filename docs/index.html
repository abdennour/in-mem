insert

Insert data in in-memory database.

insert(table: String, records: Object): (Object | Array)
Parameters
table (String) Table Name of database
records (Object) it can be one of multiple records (overloaded function)
Returns
(Object | Array): the data itself with new attributes which are "id" and "dateCreated"
Example
const {insert} = require('in-mem');
   insert('person', {firstname: "Ahmed", birth: new Date("1991/10/12")});
   insert('product', {name: "مراعي"}, {name: "iPhone"});
